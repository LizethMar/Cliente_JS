<p>Kendo UI is everything you need to build sites and apps with HTML5 &amp; JavaScript. Kendo UI Core is the free and open-source version of Kendo UI that provides access to the web's best UI widgets and key framework features, essential for developing great experiences for the web and mobile.</p>

<h2><a id="user-content-building-kendo-ui-core" class="anchor" href="#building-kendo-ui-core" aria-hidden="true"><svg aria-hidden="true" class="octicon octicon-link" height="16" version="1.1" viewBox="0 0 16 16" width="16"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Building Kendo UI Core</h2>

<p>There are two ways to get the source code for Kendo UI Core. You can either download a pre-built zip from <a href="http://www.telerik.com/kendo-ui/open-source-core">Telerik.com</a>, or build the source yourself using Gulp. The latter approach includes an option for building a distribution of Kendo UI Core that includes only the widgets and framework features required by your app.</p>

<h3><a id="user-content-installing-dependencies" class="anchor" href="#installing-dependencies" aria-hidden="true"><svg aria-hidden="true" class="octicon octicon-link" height="16" version="1.1" viewBox="0 0 16 16" width="16"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Installing Dependencies</h3>

<p>In order to build Kendo UI Core, you need to have <strong>Node.js 4.x</strong> and <strong>git 2.x</strong>.</p>

<p>For Windows you have to download and install <a href="https://git-for-windows.github.io/">Git for Windows</a> and <a href="http://nodejs.org/download/">Node.js</a> and <a href="http://landinghub.visualstudio.com/visual-cpp-build-tools">Visual C++ Build Tools</a>.</p>

<p>Mac OS users should install <a href="http://mxcl.github.com/homebrew/">Homebrew</a>. Once Homebrew is installed, run <code>brew install git</code> to install git, and <code>brew install node</code> to install Node.js.</p>

<p>Linux users can use the <a href="https://github.com/nodesource/distributions">distributions provided by NodeSource</a> or their package managers to install <code>git</code> and <code>nodejs</code>.</p>

<h3><a id="user-content-how-to-build-kendo-ui-core" class="anchor" href="#how-to-build-kendo-ui-core" aria-hidden="true"><svg aria-hidden="true" class="octicon octicon-link" height="16" version="1.1" viewBox="0 0 16 16" width="16"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>How to build Kendo UI Core</h3>

<p>Clone a copy of the repository by running</p>

<div class="highlight highlight-source-shell"><pre>git clone https://github.com/telerik/kendo-ui-core.git</pre></div>

<p>Enter the repository directory</p>

<div class="highlight highlight-source-shell"><pre><span class="pl-c1">cd</span> kendo-ui-core</pre></div>

<p>Initialize the submodule repository</p>

<div class="highlight highlight-source-shell"><pre>git submodule update --init</pre></div>

<p>Run the build script:</p>

<div class="highlight highlight-source-shell"><pre>npm run build</pre></div>

<p>The minified version of the scripts and styles of Kendo UI Core will be put in the <code>dist/</code> subdirectory.</p>

<p>If you want to create custom build or help with Kendo UI Core development, it would be better to install the <a href="https://github.com/gulpjs/gulp/blob/master/docs/getting-started.md#1-install-gulp-globally">gulp command line interface</a> as a global package:</p>

<div class="highlight highlight-source-shell"><pre>[sudo] npm install --global gulp</pre></div>

<p>Make sure you have <code>gulp</code> installed by testing:</p>

<div class="highlight highlight-source-shell"><pre>gulp --version</pre></div>

<p>Now by running the <code>gulp</code> command in the Kendo UI Core directory, you can build the full version of Kendo UI Core. This works just like the <code>npm run build</code> command:</p>

<div class="highlight highlight-source-shell"><pre>gulp build</pre></div>

<p>There are several other tasks available:</p>

<div class="highlight highlight-source-shell"><pre>gulp --tasks</pre></div>

<h3><a id="user-content-building-only-what-you-need" class="anchor" href="#building-only-what-you-need" aria-hidden="true"><svg aria-hidden="true" class="octicon octicon-link" height="16" version="1.1" viewBox="0 0 16 16" width="16"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Building only what you need</h3>

<p>The gulp <code>custom</code> task will create a custom minified file - <code>dist/kendo.custom.min.js</code>, which includes only the specified components. The following command will include only the autocomplete and dropdownlist widgets:</p>

<div class="highlight highlight-source-shell"><pre>gulp custom -c autocomplete,dropdownlist</pre></div>
